# Release 3.7.0.0

[Download](http://nerevar.github.io/jmc/pages/download/)

New in version 3.7.0.0:

- [Issue #43](https://github.com/nerevar/jmc/issues/43) Ability to log data received from the server (by default) or displayed to the user (with highlights, replacements, and displayed text applied)
- Live HTML logs: the time between commands is recorded, and the page plays back at the same speed at which the data arrived at the client
