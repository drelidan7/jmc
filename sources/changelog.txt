- Updated HTML log template — default font is Courier New
- New icon J_

New in version 3.6.0.0:
- Ability to minimize JMC to the system tray — added a "Minimize to system tray" checkbox in the settings
- Added commands:
	#hidewindow - minimizes the JMC window to the taskbar (regardless of the "Minimize to system tray" checkbox)
	#restorewindow - restores the JMC window
	#tray <hide|show> - lets you minimize / restore the JMC window to the system tray
- Changed the bat file for compatibility with Windows 7, allowing the JMC dll libraries to be registered in the system for scripts to work correctly
- Refactored logs:
	- when a log is closed, the log file name is shown
	- changed the log settings form
	- ability to keep proper html logs
	- html template for html logs:
		- built-in Fixedsys font, ability to choose a font - Fixedsys, Courier New in newer browsers
		- ability to upload the log file to the peeep.us hosting service by clicking a link
- Refactored variables. Added special variables:
	- $NOCOLOR - the "normal" color, which resets an unfinished color sequence
	- $MILLISECOND - current number of milliseconds
	- $TIMESTAMP - unix timestamp - time in seconds elapsed since January 1, 1970
- Fixes to the language file language.ini
- A "Display commands" checkbox in the settings that allows the mud's system commands to be shown on screen (and written to the log file) in square brackets
- Do not create the tabwords.txt file on exit each time if it is empty
- Maximum history size increased from 500 to 10000
