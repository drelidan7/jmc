JMC
===

[![Join the chat at https://gitter.im/nerevar/jmc](https://badges.gitter.im/nerevar/jmc.svg)](https://gitter.im/nerevar/jmc?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)

[Jaba Mud Client](http://nerevar.github.io/jmc/) is a program designed for playing multiplayer worlds — online games on the internet

[Download from the official page http://nerevar.github.io/jmc/](http://nerevar.github.io/jmc/)

Main features of the program:

- Lightweight client requiring no installation, for Windows XP/Windows Vista/Windows 7
- Beautiful HTML logs that can be shared on the internet
- Support for variables and mathematical operations, regular expressions
- Triggers, aliases, hotkeys, highlights, auto-replacements, additional buffers and windows
- High rendering speed, no eye-tiring effects ("text jitter", "blinking")
- Support for scripting languages such as JScript/VBScript/Perl
- Working with the OS clipboard; minimizing to the system tray and taskbar
- Support for numerous "profiles" to separate groups of settings
- Playback of sound files, launching any third-party applications


Intermediate (pre-release) versions can (temporarily?) be downloaded and tried at your own risk [here](https://github.com/konelav/jmc/releases).
